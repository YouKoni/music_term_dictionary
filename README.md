# music_term_dictionary
#### Java(サーブレット&JSP)、SQL、HTML/CSSを用いて「音楽記号・用語辞書システム」を作成しました。
#### 主な機能は以下の通りです。
- アカウントの新規登録、更新、削除の機能
- 音楽記号・用語の検索
- 音楽記号・用語別の解説の閲覧
  
### 開発環境
- Windows 11, version 23H2
- Eclipse 2022
- Java17/Tomcat9
- MySQL 10.4.28(MariaDBでも動作します)
